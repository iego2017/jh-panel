
### 简介

简单的Debian面板

### 环境

- Debian 11.5

### 安装

```bash
apt update && apt install -y wget && wget -O install.sh https://raw.githubusercontent.com/jianghujs/jh-panel/master/scripts/install.sh && sh install.sh
```
